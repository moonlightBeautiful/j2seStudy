使用ZXing生成/解析二维码图片
ZXing是一种开源的多格式1D/2D条形码图像处理库，在Java中的实现。
重点是在手机上使用内置摄像头来扫描和解码设备上的条码，而不与服务器通信。
然而，该项目也可以用于对桌面和服务器上的条形码进行编码和解码。目前支持这些格式：
    UPC-A and UPC-E
    EAN-8 and EAN-13
    Code 39
    Code 93
    Code 128
    ITF
    Codabar
    RSS-14 (all variants)
    RSS Expanded (most variants)
    QR Code
    Data Matrix
    Aztec ('beta' quality)
    PDF 417 ('alpha' qua
